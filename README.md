## Intended Use & Potential Applications

* Use the Maya scene and materials as a starting point for interior design or furniture visualization (e.g. a table, room setup, set-piece).
* Use the texture JPEGs as surface materials for other 3D models (granite, wood, metal, wall textures).
* As a reference material library for 3D artists: experiment with lighting, shading, rendering, or materials by applying the textures to new models.
* For learning or demonstration purposes — especially if you plan to extend this into a full-scale scene, environment, or architectural visualisation.

## How to Use

1. Clone or download the repository locally.
2. Open `Production Table.mb` in Autodesk Maya (or any compatible 3D software) that supports `.mb` files.
3. Load/apply the provided textures (`granite.jpg`, `wooden.jpg`, `steel legs.jpg`, `wall.jpg`) via the material swatches or manually for custom materials.
4. Adjust lighting, camera, and environment settings to render your desired scene, or export the models/materials for use elsewhere.
5. Optionally, reuse or remap the textures to other 3D models to build new furniture, room layouts, or visualizations.
